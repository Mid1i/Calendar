# Календарь

Реализация календаря, имеющего различные режимы выбора дат:
- Конкретная дата
- Несколько конкретных дат
- Один период времени
- Несколько периодов времени

## Стек технологий:
- TypeScript
- Vue.js
- Pinia
- SCSS
